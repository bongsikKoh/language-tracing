# Language Tracing

휴대폰, 태블릿, 데스크톱 브라우저에서 사용할 수 있는 터치 기반 언어 따라쓰기 웹앱입니다.

현재 첫 버전은 한글 단어 학습에 맞춰져 있으며, 이후 영어와 한문 같은 다른 언어 모드로 확장할 수 있도록 프로젝트명을 `language-tracing`으로 정리했습니다.

## 기능

- 시작 화면에서 `시작`을 누르면 랜덤 단어가 표시됩니다.
- 그림과 회색 음영 한글을 함께 보여줍니다.
- 손가락, 펜, 마우스로 회색 글자를 따라 쓸 수 있습니다.
- 일정 이상 따라 쓰면 효과음이 재생되고 단어를 천천히 읽어줍니다.
- `다시 쓰기`, `읽어주기`, `다음 단어`를 지원합니다.
- PWA 설정이 포함되어 브라우저에서 홈 화면에 추가할 수 있습니다.

## 실행

정적 파일만으로 구성되어 있어 `index.html`을 브라우저로 열면 됩니다.

로컬 서버로 확인하려면:

```powershell
python -m http.server 5500
```

그 다음 브라우저에서 `http://localhost:5500/language-tracing/`을 엽니다.
